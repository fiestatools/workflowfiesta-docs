# CLI

The WorkflowFiesta CLI brings AI-powered workflow automation directly to your terminal. Chat with agents, switch between them, and manage conversations — all without leaving the command line.

{% hint style="info" %}
Use the sidebar to navigate the CLI reference: [Installation](cli/installation.md), [Configuration](cli/configuration.md), [Commands](cli/commands.md), and [Agent Interactions](cli/agent-interactions.md).
{% endhint %}

## Requirements

- macOS, Linux, or Windows
- A WorkflowFiesta account with an API token (Settings → API Keys)

## What you can do

- Chat with any agent in your org from the terminal
- Switch agents mid-conversation with `/agent`
- Browse conversation history with `/history`
- Handle credential prompts, OAuth flows, and MCP connections inline — no browser switching required
